# Task 1: Sales Dataset Dashboard Creation

This project focuses on building an interactive and dynamic sales performance dashboard using MS Excel.

### Key Features & Elements:
* **Key Performance Indicators (KPIs):** Displays crucial business metrics including Total Sales, Total Profit, and Total Orders.
* **Trend Tracking:** Utilizes line charts to monitor order distributions and trends across various years and months.
* **Categorical Performance:** Breaks down transaction volumes and profitability across customer Segments, product Categories, and Shipping Modes.
* **Interactive Slicers:** Equipped with dynamic filtering options for Region, Category, Month, and Year to enable user-driven insights.
