# Task 1: Excel Sales Dashboard

This project focuses on cleaning, analyzing, and visualizing a Superstore Sales dataset using Microsoft Excel to build an interactive performance dashboard.

### Project Objective:
The primary goal is to transform raw sales transactions into actionable business insights by tracking key performance indicators and identifying high/low performing areas.

### Key Features & Requirements Implemented:
* **Data Cleaning & Analysis:** Processed and structured raw sales data to prepare it for robust analysis.
* **Pivot Tables & Charts:** Built dynamic summaries to break down transaction volume and performance.
* **Core Metrics Visualized:**
  * **Total Sales & Profit:** High-level overview of overall business revenue and margins.
  * **Profit by Category:** In-depth evaluation of profitability across different product groups.
  * **Sales Trends Over Time:** Line graphs tracking order distribution across years and months to detect patterns.
* **Conditional Formatting:** Applied visual formatting rules to instantly highlight outstanding high-performers and critical low-performing segments.
* **Interactive Slicers:** Included user-friendly filtering options for quick cross-sectional data discovery.
